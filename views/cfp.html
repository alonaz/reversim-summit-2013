Call For Papers (Deadline: Jan 1st)

Process:

  1. Submit your 'paper' (it's actually a talk or a lab). Deadline: Jan 1 2013
  2. Committee chooses the selected papers and creates an agenda
  3. We contact you and send date/time confirmation
  4. You prepare the slides/lab/rest of the work

We want you on board. Please see the full instructions here.

require 'speaker'
require 'israel'

module Israel
  module Summit
    def call_for_papers(speaker, subject)
      apply(speaker) if ["DevOps, NoOps, CD, Dev Culture",
                         "Software Craftsmanship",
                         "Mobile dev",
                         "Web dev",
                         "Product, Lean Startup, UX"
                        ].include(subject)
                        and (speaker.available?(Date.new(2013, 2, 18))
                           or speaker.available?(Date.new(2013, 2, 19)))
    end
    def apply(speaker)
      # We prefer github, but also settle for goo-ol emails
      if speaker.can_github?
        speaker.make_pull_request('/rantav/reversim-summit-2013')
      else
        speaker.send_mail('rantav@ the gmail')
    end
  end
end
